String

  • python

字符串是一种常见数据结构。

Spilt()函数

对于绝大部分字符串，使用python自带的Spilt()函数就足以完成预期的功能。

val = 'a,b, guido'
val.split(',')
#这就是最简单的例子
pieces = [x.strip() for x in val.split(',')]
'::'.join(pieces)
#这用来添加中间的：：
#另外还有这些函数可以使用
val.index(',')
val.find(',')

正则表达式

正则表达式（通常称作 regex）提供了一种灵活的在文本中搜索或匹配字符串模式的方式。正则表达式是根据正则表达式语言编写的字符串。Python 内置的 re 模块负责对字符串应用正则表达式。

import re
regex = re.compile('\s+')
regex.split(text)
['foo', 'bar', 'baz', 'qux']
# 如果只希望得到匹配 regex 的所有模式，则可以使用 findall 方法
regex.findall(text)
[' ', '\t ', ' \t']

match 和 search 跟 findall 功能类似。findall 返回的是字符串中所有的匹配项，而 search 则只返回第一个匹配项。match 更加严格，它只匹配字符串的首部。
