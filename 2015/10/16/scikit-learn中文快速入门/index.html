scikit-learn中文快速入门

  • scikit-learn

最近再看scikit-learn的文档，决定把一些内容和自己的理解记录下来。本人英文水平十分有限，加之翻译时有部分遗漏，请谅解。

原文地址：http://scikit-learn.org/stable/tutorial/basic/tutorial.html

机器学习问题的设定

一般来讲，机器学习可以看做思考n个样本的数据，然后尝试求解未知的数据。例如对于，对于一个多维的问题，我们可能需要考虑一系列的特征。
我们把机器学习分成这样几类：
监督学习：这些数据会带来特定的结果。而监督学习有分为以下几类：

分类：分类是样本属于两个或更多的类别。我们通过学习已经表好label的数据来预测未标记的数据。手写字的识别就是典型的分类问题。另外分类是一种离散（相对于连续），他只能限制于样本所提供的数据。

回归：如果期待的输出是一系列连续的变量，那么这个任务叫做回归。一个例子就是用年龄和体重预测salmon的长度。

非监督学习：这种学习有一系列的X值没有相应的目标值。这种问题的学习目标是探索数据的群组。我们叫做聚类，也是在输入空间上探索数据。正如密度估计，或者为了可视化把数据从高维降到两维或三维。

载入数据集

scikit提供了一些标准的数据集，例如iris，digits给分类问题，波士顿房价问题给回归。
接下来我们使用python的交互环境，载入这两个数据集：

$ python
>>> from sklearn import datasets
>>> iris = datasets.load_iris()
>>> digits = datasets.load_digits()

数据集使用一种类似字典的数据结构，储存着数据。这时使用.data成员来储存这是一个数组（ n_samples, n_features），对于监督学习，需要一个或者变量作为.target成员。

模型持久化

我们使用python库的pickle来实现持久化。

>>> from sklearn import svm
>>> from sklearn import datasets
>>> clf = svm.SVC()
>>> iris = datasets.load_iris()
>>> X, y = iris.data, iris.target
>>> clf.fit(X, y)  
SVC(C=1.0, cache_size=200, class_weight=None, coef0=0.0, degree=3, gamma=0.0,
  kernel='rbf', max_iter=-1, probability=False, random_state=None,
  shrinking=True, tol=0.001, verbose=False)

>>> import pickle
>>> s = pickle.dumps(clf)
>>> clf2 = pickle.loads(s)
>>> clf2.predict(X[0])
array([0])
>>> y[0]
0

对于特殊的例子，我们可以使用joblib来替换，对于大数据量效率更高。

>>> from sklearn.externals import joblib
>>> joblib.dump(clf, 'filename.pkl')
