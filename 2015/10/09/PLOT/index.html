PLOT

  • python

绘图是数据分析工作中最重要的任务之一，是探索过程的一部分，例如，帮我们找出异常值，必要的数据转换、得出有关模型的 idea 等。

matplotlib工具包

matplotlib 的图像都位于 Figure 对象中。你可以使用 plt.figure 创建一个新的Figure。

import matplotlib.pyplot as plt
fig = plt.figure()
fig.show()

不能通过空Figure绘图，必须使用add_subplot创建subplot。

ax1 = fig.add_subplot(2,2,1)
ax2 = fig.add_subplot(2,2,2)
ax3 = fig.add_subplot(2,2,3)

这样就添加三个子图。

其余的具体颜色形状和图例都和MATLAB的绘图十分相似，我就不详细叙述了。

pandas中绘图函数

线形图

对于Series和DataFrame都有一个plot方法，默认都是线形图。

df.plot()
s.plot()
plt.show()

柱状图

在生成线性图的代码中加上 kind = ‘bar’（垂直柱状图）或 kind = ‘barh’（水平柱状图）即可生成柱状图。

data = Series(np.random.rand(16),index = list('abcdefghijklmnop'))
data.plot(kind='bar',ax=axes[0],color='b',alpha=0.7)
