Support Vector Machines

  • ML

这篇文章主要介绍了支持向量机在scikit-learn包的使用。原文地址：http://scikit-learn.org/stable/modules/svm.html

分类问题

SVC，NuSVC和线性SVC的输入是两个数组（X,Y）X是训练样本,Y是分类的标签（字符串或者实数）。

>>> from sklearn import svm
>>> X = [[0, 0], [1, 1]]
>>> y = [0, 1]
>>> clf = svm.SVC()
>>> clf.fit(X, y)  
SVC(C=1.0, cache_size=200, class_weight=None, coef0=0.0, degree=3,gamma=0.0, kernel='rbf', max_iter=-1, probability=False, random_state=None,shrinking=True, tol=0.001, verbose=False)

>>> clf.predict([[2., 2.]])
array([1])

支持向量机依赖于训练数据的子集，被称作支持向量。这些支持向量可以被发现在成员变量supportvectors,support_ 和 n_support

>>> # get support vectors
>>> clf.support_vectors_
array([[ 0.,  0.],
   [ 1.,  1.]])
>>> # get indices of support vectors
>>> clf.support_ 
array([0, 1]...)
>>> # get number of support vectors for each class
>>> clf.n_support_ 
array([1, 1]...)

多类别的分类

SVC和NuSVC使用了一对一对于多重分类问题。对于N个类别存在N*(N-1)/2个分类器。每两个训练集有一个分类器。

>>> X = [[0], [1], [2], [3]]
>>> Y = [0, 1, 2, 3]
>>> clf = svm.SVC()
>>> clf.fit(X, Y) 
SVC(C=1.0, cache_size=200, class_weight=None, coef0=0.0, degree=3,
gamma=0.0, kernel='rbf', max_iter=-1, probability=False, random_state=None,
shrinking=True, tol=0.001, verbose=False)
>>> dec = clf.decision_function([[1]])
>>> dec.shape[1] # 4 classes: 4*3/2 = 6
6

另一方面，线性SVC使用了one-vs-the-res策略，因此训练n个类别，其实只是两个类别，只有一个模型被训练。

>>> lin_clf = svm.LinearSVC()
>>> lin_clf.fit(X, Y) 
LinearSVC(C=1.0, class_weight=None, dual=True, fit_intercept=True,
 intercept_scaling=1, loss='squared_hinge', max_iter=1000,
 multi_class='ovr', penalty='l2', random_state=None, tol=0.0001,
 verbose=0)
>>> dec = lin_clf.decision_function([[1]])
>>> dec.shape[1]
4

分割超平面在不平衡类

原文地址：http://scikit-learn.org/stable/auto_examples/svm/plot_separating_hyperplane_unbalanced.html
这里给出了一个使用SGD（stochastic Gradient Descent）梯度随机下降来代替SVC。

print(__doc__)

import numpy as np
import matplotlib.pyplot as plt
from sklearn import svm
#from sklearn.linear_model import SGDClassifier

# we create 40 separable points
rng = np.random.RandomState(0)
n_samples_1 = 1000
n_samples_2 = 100
X = np.r_[1.5 * rng.randn(n_samples_1, 2),
      0.5 * rng.randn(n_samples_2, 2) + [2, 2]]
y = [0] * (n_samples_1) + [1] * (n_samples_2)

# fit the model and get the separating hyperplane
clf = svm.SVC(kernel='linear', C=1.0)
clf.fit(X, y)

w = clf.coef_[0]
a = -w[0] / w[1]
xx = np.linspace(-5, 5)
yy = a * xx - clf.intercept_[0] / w[1]


# get the separating hyperplane using weighted classes
wclf = svm.SVC(kernel='linear', class_weight={1: 10})
wclf.fit(X, y)

ww = wclf.coef_[0]
wa = -ww[0] / ww[1]
wyy = wa * xx - wclf.intercept_[0] / ww[1]

# plot separating hyperplanes and samples
h0 = plt.plot(xx, yy, 'k-', label='no weights')
h1 = plt.plot(xx, wyy, 'k--', label='with weights')
plt.scatter(X[:, 0], X[:, 1], c=y, cmap=plt.cm.Paired)
plt.legend()

plt.axis('tight')
plt.show()

关于如何把权重合理的表现出来，这里有一个例子：http://scikit-learn.org/stable/auto_examples/svm/plot_weighted_samples.html#example-svm-plot-weighted-samples-py

算法复杂度

支持向量机是一个非常强大的工具，但是它要求的存储空间随着训练集增大迅速增加。到

关于参数的使用和技巧

  • 避免数据复制：Avoiding data copy: For SVC, SVR, NuSVC and NuSVR, if the data passed to certain methods is not C-ordered contiguous, and double precision, it will be copied before calling the underlying C implementation. You can check whether a give numpy array is C-contiguous by inspecting its flags attribute

  • 对于linearSVC建议在使用double是改用SGD

  • 核缓存，如果内存足够建议改变cache_size的默认值 例如500MB或者1000MB。

  • 设置参数C，如有你的数据有很大的噪音，你应该减小C
  • 推荐规整你的数据，例如输入X在[0,1]或者[-1,+1]之间，或者标准化它让他平均值为0，记住相同的规整应该也在你的测试数据中进行
  • 参数nu对于NuSVC…
  • 在SVC中如果数据不平衡，使用set_weight=’auto’
  • linearSVC每次会使用一个随机数选择特征，因此对于相同的数据，结果可能会不同。
  • Using L1 penalization as provided by LinearSVC(loss=’l2’, penalty=’l1’, dual=False) yields a sparse solution, i.e. only a subset of feature weights is different from zero and contribute to the decision function. Increasing C yields a more complex model (more feature are selected). The C value that yields a “null” model (all weights equal to zero) can be calculated using l1_min_c.
