第四章 分类器

第四章 分类器

简介

分类器在一系列情况下都十分重要。例如如果我们想自动化完成一些决策过程，我们可以使用分类器。

使用决策树来分类

这一节我们来使用决策树来分类，这是一个非常好的模型，因为它很好理解。一旦训练好了，这里通常使用SQl语句，这意味着很多人可以使用它。

准备

我们首先还是制作我们需要的数据：

>>> from sklearn import datasets
>>> X, y = datasets.make_classification(n_samples=1000, n_features=3,
 n_redundant=0)

怎样去做

使用决策树十分简单，我们首先引入对象，然后拟合模型。

>>> from sklearn.tree import DecisionTreeClassifier
>>> dt = DecisionTreeClassifier()
>>> dt.fit(X, y)
DecisionTreeClassifier(compute_importances=None, criterion='gini',    max_depth=None, max_features=None,    max_leaf_nodes=None, min_density=None,min_samples_leaf=1, min_samples_split=2,random_state=None, splitter='best')
preds = dt.predict(X)
(y == preds).mean()
    1.0

正如你可以看见的，我们来看看我们可以调整的参数。首先，如果你看着这个对象，他有这样几个关键的参数，可以用来影响。
第一个是 max_depth 这是一个非常重要的参数， 这确定个多少个分支。我们来看看我们使用不同的max_depth会带来怎样的效果：

n_features=200
X, y = datasets.make_classification(750, n_features,
n_informative=5)
import numpy as np
training = np.random.choice([True, False], p=[.75, .25],size=len(y))
accuracies = []
for x in np.arange(1, n_features+1):
dt = DecisionTreeClassifier(max_depth=x)
dt.fit(X[training], y[training])
preds = dt.predict(X[~training])
accuracies.append((preds == y[~training]).mean())
import matplotlib.pyplot as plt
f, ax = plt.subplots(figsize=(7, 5))
ax.plot(range(1, n_features+1), accuracies, color='k')
ax.set_title("Decision Tree Accuracy")
ax.set_ylabel("% Correct")
ax.set_xlabel("Max Depth"）

我们可以看出来我们应该使用比较小的值我们用最大值15来看看这个结果：

N = 15
import matplotlib.pyplot as plt
f, ax = plt.subplots(figsize=(7, 5))
ax.plot(range(1, n_features+1)[:N], accuracies[:N], color='k')
ax.set_title("Decision Tree Accuracy")
ax.set_ylabel("% Correct")
ax.set_xlabel("Max Depth")

这样我们可以看见明显的下降的趋势，1-3之间应该是最好的的。
我们接下来看看compute_importances参数，它其实更象随机森林的方法，但是如果你使用0.16以后的版本你会看见它。

dt_ci = DecisionTreeClassifier(compute_importances=True)
